“长图缩略彩蛋”小工具 - 轻量prd（移动端优先，纯前端，vercel 部署）
项目概述

最终效果参考：https://imagehost.qianzhu.online/api/rfile/demo1.jpg（作为目标观感与交互的参考基线）。

项目名称：PeekSwap Lite（可改）。

一句话目标：移动端网页中，用户上传两张图（微信会话“封面图A”与“点开后效果图B”），在线一键合成为一张“超长图”，在微信会话预览仅显示该长图中部的A，点开查看时展示完整画面以B为主，形成“展开有彩蛋”的体验。

定位：大学生/社团/课堂的小型创意工具，纯前端、免登录、免下载，Vercel 部署。

合规声明：仅用于创意彩蛋、公益/活动预告与教育科普；禁止用于误导、冒充、违规传播。

项目名称：PeekSwap Lite（可改名）

一句话目标：让用户在移动端网页里上传两张图（微信封面 / 点击后效果图），一键在浏览器内合成为一张“超长图”，满足在微信会话内预览居中裁切看到“封面图”，点开查看时看到“效果图为主”的视觉反转。

定位：课堂/社团/同学间的创意玩具，纯前端、免登录、免下载、移动端优先，一键在 Vercel 托管上线。

合规声明：仅用于创意彩蛋、活动预告等正当用途；不得用于欺骗、冒充或误导。

Must Have（硬性需求）

单页应用（SPA），移动端优先（375–430px 视口良好），无需登录/注册。

两张图片上传：

A（封面图）：在会话预览中被看到。

B（效果图）：点开后主要内容。

所见即所得预览：

模拟“会话预览窗”：仅显示合成后长图的中部居中区域（可根据参数动态变化）。

“完整长图预览”：展示最终生成的整张图（可滚动）。

参数仅 2 个：

封面区高度（范围 15%–40%，默认 28%）。

提示水印开关（默认开启，右下角“展开有彩蛋”）。

一键合成与导出：浏览器内生成单张图片（PNG 或 JPEG），提供保存指引（移动端长按保存/桌面端下载）。

本地处理：图像处理全在本地浏览器中完成，不上传服务器。

Vercel 部署：提供 README 与基础脚本，保证一键导入即可上线。

Nice to Have（可选）

自动压缩：导出前将长边限制在 4096–6000px（缓解体积）。

比例自适应：若图A/B 宽高比差异大，给出“等比填充（留边）/裁切适配”二选一。

一键复制文案：复制“建议配文：‘点开有彩蛋’”。

合成原理（实现简述）

平台假设：微信会话列表/聊天窗口预览通常取图片居中区域作为缩略图；用户点击后才查看整张图片。此工具通过构造超长图，将 A 放在中部“封面区”，将 B 作为上下主体区域，从而实现“预览看A、展开看B”。

合成思路（用语义描述，不涉及代码）：

以效果图 B 的宽度作为输出基准，得到目标显示宽度与对应高度；

依据“封面区高度百分比”计算中部区域的高度与位置；

画面从上到下分为三段：上段（来自 B 的上部内容）、中段（放置 A，居中对齐）、下段（来自 B 的下部内容）；
