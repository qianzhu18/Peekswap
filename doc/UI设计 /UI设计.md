PeekSwap Lite “搞事彩蛋实验室” UI 方案（最终版）

## 1. 定位与体验主线
- 单页 Web 服务，纯前端实现，所有图片处理在浏览器内完成，可直接部署到 Vercel（静态站点或任意前端框架均可）。
- 设计语言走“好玩整蛊、鼓励搞事”路线，目标用户为学生/社团，语气调皮、不谈版权或水印。
- 移动端优先（375–430px），保证核心流程一屏内串联；平板及桌面仅做增强适配，完整体验仍以移动端为基准。
- 用户仅需上传两张图（封面图 A、效果图 B），实时预览缩略效果并调整封面区高度后，一键合成导出超长图。

## 2. 页面结构（从上到下）
1. **粘性顶部栏**  
   - 左：品牌标题“PeekSwap Lite · 搞事模式” + 口号“点开有惊、截屏有梗”。  
   - 右：按钮“玩家须知”，点开底部抽屉展示使用提醒、整蛊守则（无版权/水印字样）。
2. **预览区**  
   - Tab 切换“好友预览”“彩蛋全貌”。  
   - “好友预览”模拟聊天缩略窗，仅显示长图中部；“彩蛋全貌”为 Scroll Snap 容器，可上下滑动。
3. **上传区**  
   - 两张卡片：“先放封面哄他”（图 A）与“再塞彩蛋炸他”（图 B），支持点击选择 / 移动端调用相册 / 桌面拖拽。  
   - 上传后显示缩略图、文件名、像素尺寸，并提供替换和删除操作。
4. **参数区**  
   - 唯一参数：封面区高度滑杆（15%–40%，默认 28%），刻度标签“稳健”“标准”“爆梗”，旁侧显示实时百分比。  
   - 若检测到 A/B 宽高比差异较大，弹出“等比留边 / 智能裁切”提示面板（可选项，属增强体验，不新增全局参数）。
5. **调皮提示区**  
   - 折叠卡片/小贴士形式，列出“整蛊守则”“好友反应建议”“截图发圈灵感”等趣味文案，加强产品语言。
6. **底部固定操作条**（移动端 48px 高）  
   - 按钮从左到右：`刷新槽点`（重绘预览）、`上线整活`（触发合成）、`拷走彩蛋`（导出指引）。  
   - 合成完成后，在操作条上方弹出“整蛊成功卡”，展示缩略图、分辨率、文件大小与“复制推荐文案”按钮。

## 3. 核心流程交互
- **上传**：即时校验文件类型、尺寸与大小；出错提示用调侃语气（例：“太糊啦，他可能认不出你”）。上传中展示骨架+进度动画。
- **实时预览**：滑杆变化以 150ms 动画过渡刷新截取范围；Tab 切换伴随彩纸粒子特效。确保“好友预览”与“彩蛋全貌”同步更新。
- **合成与导出**：采用 Canvas/OffscreenCanvas 本地合成，展示进度条（“彩蛋装填 73%...”）；完成后触发“彩蛋装填完毕！” Toast 与短震反馈（支持设备）。  
  - 移动端提示“长按图片保存 / 截屏也行”；桌面端提供直接下载按钮与拖拽保存说明。
- **文案语气**：始终保持整蛊幽默风格，如按钮副文案“别怂，去炸群”；提示卡“记得看朋友反应、截图留念”等。

## 4. 视觉与动效规范
- **色彩**：背景渐变 #1C1542 → #FF2E90；强调色酸性黄 #FFE45C 配薄荷青 #2FF0B5；辅助用明亮蓝紫点缀。
- **字体**：标题使用手写感/泡泡体 24–26px（黑色或深紫描边）；正文采用 `PingFang SC` / `Inter` 15–16px，字重 Medium；桌面端自动放大 2px。
- **组件**：卡片白色或浅色毛玻璃，圆角 12px，轻微霓虹阴影；按钮 48px 高（移动端），桌面端保留触控尺寸并加入 Hover 发光。
- **图形**：漫画风插画（笑脸、爆炸云、聊天气泡）；Tab 顶部贴纸如“好友被整蛊预览”“整活进行中”。
- **动效**：  
  - Tab 切换：彩纸粒子 + 200ms 淡入。  
  - 滑杆拖动：封面预览短暂出现“???” 覆盖后揭示。  
  - 按钮点击：弹簧缓动与色彩闪烁；合成成功卡伴随彩带下落。

## 5. 响应式策略
- **移动端（≤430px）**：单列布局，模块全宽排列，底部操作条固定；滚动区域在操作条之上终止。
- **平板/桌面（≥768px）**：页面改为左右分栏，左侧 60% 主要展示双预览卡，右侧垂直展示上传、参数与提示；底部操作条变为右下角悬浮按钮组（同样文案与动效）。
- **桌面增强**：支持拖拽上传、文件名悬浮显示完整信息、快捷键提示（如 `⌘+Z` 撤销上一次上传）；保持触控友好的交互尺寸以兼容触摸屏笔记本。

## 6. 内容示例
- Tab 文案：“好友预览”“彩蛋全貌”
- 按钮文案：“刷新槽点”“上线整活”“拷走彩蛋”
- Toast 示例：“彩蛋装填完毕！发出去，看他们表情！”
- 提示卡片：“只整熟悉的人”“别越界才好玩”“记得截图留证据”
